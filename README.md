# 🎓 Student Dashboard

Application web permettant de gérer des étudiants et leurs notes.

---

## 🚀 Objectif

Créer une application dynamique en **JavaScript** pour :
- ajouter des étudiants
- gérer leurs notes
- calculer leurs moyennes
- afficher les données de manière interactive

---

## 🛠️ Technologies utilisées

- HTML
- CSS
- JavaScript (DOM, fonctions, tableaux, objets)

---

## ⚙️ Fonctionnalités

### 👤 Gestion des étudiants
- Ajout d’un étudiant
- Vérification des doublons
- Validation des champs

### 📝 Gestion des notes
- Ajout de notes (entre 0 et 20)
- Association des notes à un étudiant

### 📊 Affichage dynamique
- Tableau des étudiants
- Affichage des notes
- Calcul automatique des moyennes

### 🔍 Recherche
- Filtrage des étudiants en temps réel avec `.filter()`

### 🏆 Meilleur étudiant
- Identification de l’étudiant avec la meilleure moyenne
- Mise en évidence dans l’interface

---

## 🧠 Concepts JavaScript utilisés

- Manipulation du DOM
- Tableaux et objets
- Boucles (`for`)
- Fonctions
- Conditions
- Gestion des erreurs avec `try...catch`
